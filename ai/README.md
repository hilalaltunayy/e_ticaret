# AI Workspace

This folder is the AI working layer of the e-commerce project.

It stores project context, architecture rules, task definitions, reusable skills, and prompt templates for AI-assisted development.

## Main Files

- project.md: Project purpose and scope
- architecture.md: Technical architecture and layering
- rules.md: Non-negotiable development rules
- current_state.md: Current implementation state
- decisions.md: Important architectural decisions
- backlog.md: Future work and pending items

## Folders

- epics/: Large feature areas
- tasks/: Small executable development tasks
- skills/: Reusable AI working methods
- prompts/: Reusable Codex prompt templates

## Usage

Before giving a task to Codex:

1. Read current_state.md
2. Read rules.md
3. Select the related task file
4. Select required skill files
5. Use a prompt template from prompts/
6. Ask Codex to return changed files, summary, commands, and test steps
