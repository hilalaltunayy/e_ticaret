# Oracle MCP Runtime

This folder is reserved for future local Oracle MCP runtime files.

No secrets belong here.

`app/` remains read-only for Oracle runtime planning and future Oracle MCP usage unless a separate explicit implementation policy says otherwise.

Dockerfile and Docker Compose files are not created yet.

MCP server code is not created yet.

The Docker seccomp warning must be reviewed before Dockerfile or Docker Compose hardening.

Any future `.env.example` must use placeholders only.

Do not place real `.env`, wallet files, credentials, API keys, private keys, passwords, certificates, or tokens in this folder.
